# Web Scraping with Beautiful Soup Course

This repository contains the materials for a web scraping course using Python and the Beautiful Soup library.

## Course Structure

The course is divided into several lessons, each in its own numbered directory. Each lesson directory contains:

- `scraper.py`: The Python script for the lesson.
- `notes.md`: Explanations and notes for the lesson.

## Getting Started

1.  **Install Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

2.  **Follow the Lessons:**
    Start with lesson `01-Introduction` and proceed in order.
